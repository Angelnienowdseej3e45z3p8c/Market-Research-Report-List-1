<p><ul><li>Pit Set Interface</li><li>Non-Pit Set Interface</li></ul></p>
<p><p>Meter Transceiver Unit (MXU) Market Types include Pit Set Interface and Non-Pit Set Interface. The Pit Set Interface MXUs are installed below ground level inside a meter pit, providing protection from external elements. On the other hand, Non-Pit Set Interface MXUs are installed above ground, usually on the side of a building or pole. Both types serve the purpose of facilitating communication between smart water meters and the utility company, but differ in their installation methods and exposure to environmental hazards.</p></p>
