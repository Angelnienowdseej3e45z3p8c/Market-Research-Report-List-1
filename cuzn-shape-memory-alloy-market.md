<p><ul><li>One-Way Memory Effect</li><li>Two-Way Memory Effect</li></ul></p>
<p><p>CuZn Shape Memory Alloy Market Types include One-Way Memory Effect and Two-Way Memory Effect. One-Way Memory Effect refers to alloys that can recover their original shape upon heating but only once. This market type is mainly used in applications where a one-time shape recovery is required. On the other hand, Two-Way Memory Effect alloys can recover their original shape multiple times when subjected to specific conditions. This market type is preferred in applications that require repetitive shape changes.</p></p>
